<objective>
Build the immutable-baseline what-if scenario engine. Add `scenario_mod` with `type(scenario_t)`
and `apply_scenario` (one driver per scenario, clamped to [0,1]), and wire the driver to run
baseline + "add trees" + "more concrete" by copy-then-mutate, reporting per-cell and
city-average feels change versus the same baseline at the same timesteps.

Purpose: SCEN-01 (baseline + >=1 add-trees + >=1 more-concrete without mutating the baseline,
via Fortran's deep-copy-on-assignment of the allocatable `cells(:,:)`, D-07) and SCEN-02
(per-cell + city-average delta vs baseline, apples-to-apples at the same timestep, D-08).
The mutation magnitudes (`add_trees_delta`, `concrete_delta`) come from `coeffs.nml` (added in
03-01, D-05). "more concrete" raises `building` only and does NOT flip `is_urban` (D-06).
Output: `scenario_mod`, a scenario test suite, and the scenario-loop driver with the minimal
per-scenario city-average delta console line (D-12).
</objective>
