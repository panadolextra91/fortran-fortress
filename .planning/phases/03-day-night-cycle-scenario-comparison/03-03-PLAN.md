automated test that fails the build unless `gap_predawn > gap_afternoon` for identical baseline

verified by an automated assertion, and night-edge cells still return sane near-air-temp
heat-index values (re-verifies HEAT-02). The HARD assertion is the PAIRWISE
`gap_predawn > gap_afternoon` ONLY (D-10) — NOT "pre-dawn is the global maximum gap across all
timesteps" (with these defaults early-evening can numerically edge pre-dawn because a warmer
base activates more heat-index amplification; 03-RESEARCH Anti-Pattern / Pitfall 2). The
magnitude target (~3-8 C peak night) is a SOFT, warn-only check that MUST NOT fail the build
(D-10, honoring Phase-2 D-11 rank/sign-not-absolute). Invariant tests use in-test synthetic
archetypes (D-11), not the seed data file.
Output: `summary_mod`, `test_gap.f90` (HARD + soft), and the driver gap line (D-12).
